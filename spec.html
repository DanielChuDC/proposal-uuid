Introduction

The standard module _@std:uuid_ exposes an API for generating character encoded Universally Unique IDentifiers (UUID), based on the IETF rfc4122 proposal

import {v4} from uuid

1. Let _uuid_ be the result of calling v4(). 1. let _clockSeqHiAndReserved_ be the binary value 0b10000000. 1. let _timeHiAndVersion_ be the binary value 0b01000000. 1. let _timeLow_ be the value of crypto.getRandomValues(new Uint32Array(1))[0]. 1. let _timeMid_ be the value of crypto.getRandomValues(new Uint16Array(1))[0]. 1. let _clockSeqLow_ be the value of crypto.getRandomValues(new Uint8Array(1))[0]. 1. let _byte_ be the value of crypto.getRandomValues(new Uint8Array(6))
